# @jqvelin/eslint-config

ESLint configs for plain TypeScript and React TypeScript projects.

## Available configs

### `@jqvelin/eslint-config/typescript`

Use this for non-React TypeScript projects.

```js
export { default } from '@jqvelin/eslint-config/typescript';
```

### `@jqvelin/eslint-config/react-typescript`

Use this for React projects written in TypeScript.

```js
export { default } from '@jqvelin/eslint-config/react-typescript';
```
